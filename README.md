# 🎓 VeeLearn

**Bangladesh's leading online tech learning platform — Learn. Build. Get Hired.**

---

## 🚀 About

VeeLearn is a modern, fully responsive ed-tech landing page built with pure HTML, Tailwind CSS, and vanilla JavaScript. It showcases a complete online learning platform UI — featuring course listings, instructor profiles, pricing plans, testimonials, and more. Built as a frontend project, with a full **MERN Stack** integration coming soon.

## 🎯 Key Features

- **Hero Section** — Live session badge, search bar, trending tags & CTA buttons
- **Course Categories** — Web Dev, UI/UX, AI, Mobile, Cloud, Cybersecurity
- **Upcoming Courses** — Pre-enrollment cards with progress bars & early-bird pricing
- **Popular Courses** — Filterable grid with hover preview, ratings & discount badges
- **Instructor Profiles** — Cards with animated spin rings, stats & skill tags
- **Testimonials** — Real student success stories
- **Pricing Plans** — Starter (Free), Pro Learner & Team/Business tiers
- **Responsive Navbar** — With mobile hamburger menu
- **Announcement Bar** — Dismissible offer banner
- **Footer** — Newsletter signup, contact info, social links & app store buttons
